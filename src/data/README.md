# 关键词配置文件说明

## 📋 文件位置

`src/data/keywords.json` - 搜索关键词配置文件

## 🔧 使用方法

### 修改关键词列表

直接编辑 `src/data/keywords.json` 文件，修改 `keywords` 数组中的关键词：

```json
{
  "keywords": [
    "耕地地力",
    "惠农",
    "大豆玉米带状复合种植",
    // ... 添加或修改关键词
  ]
}
```

### 重新构建

修改后需要重新构建扩展：

```bash
npm run build
```

### 重新加载扩展

1. 打开 Chrome 扩展管理页面：`chrome://extensions/`
2. 找到 "Data Hunter Pro" 扩展
3. 点击刷新按钮 🔄
4. 刷新目标网页
5. 重新点击插件中的"识别搜索框并添加关键词快捷选择"按钮

## 📝 配置说明

### JSON 结构

```json
{
  "keywords": [
    "关键词1",
    "关键词2",
    ...
  ],
  "version": "1.0.0",
  "lastUpdated": "2025-01-27",
  "description": "农业补贴政策搜索关键词列表"
}
```

### 字段说明

- `keywords`: 关键词数组，用于搜索框快捷选择
- `version`: 配置文件版本号（可选）
- `lastUpdated`: 最后更新时间（可选）
- `description`: 配置描述（可选）

## ⚠️ 注意事项

1. **JSON 格式**：确保 JSON 格式正确，每个关键词用双引号包裹，用逗号分隔
2. **编码**：文件使用 UTF-8 编码，支持中文关键词
3. **构建**：每次修改后必须运行 `npm run build` 才能生效
4. **重新加载**：修改后需要重新加载扩展并刷新目标网页

## 🔄 工作流程

1. 修改 `src/data/keywords.json`
2. 运行 `npm run build`
3. 在 Chrome 扩展管理页面重新加载扩展
4. 刷新目标网页
5. 使用插件功能

## 💡 提示

- 关键词数量没有限制，但建议控制在 50 个以内以保证良好的用户体验
- 关键词会按顺序显示在搜索框上方的按钮中
- 如果 JSON 文件加载失败，会使用代码中的默认关键词作为后备
