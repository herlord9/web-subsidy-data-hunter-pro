# Data Hunter Pro

一个专业的网页数据猎取工具浏览器扩展，使用现代化的技术栈实现。

## ✨ 主要特性

- 🔍 **智能抓取** - 自动识别网页列表/表格数据，支持多种网站格式
- 📊 **灵活选择** - 智能检测多个数据容器，用户可选择目标列表
- 📋 **多格式导出** - 支持 CSV、JSON 格式导出
- ☑️ **行选择功能** - 可选择特定行进行导出
- 🌏 **地域数据** - 支持省市区下拉选择，方便地域数据分类
- 🔗 **API 集成** - 可将数据直接导出到后端 API
- 💾 **抓取器管理** - 创建、编辑、删除抓取器配置
- ⚙️ **高级选项** - 自动滚动、分页处理、等待时间控制
- 🌍 **多语言支持** - 支持英文和中文界面

## 🚀 快速开始

### 安装
```bash
# 克隆项目
git clone <repository-url>
cd easy-scraper-clone

# 安装依赖
npm install

# 构建扩展（默认使用测试环境）
npm run build
```

### 环境配置
插件支持三种环境：

- **测试环境**（默认）: `npm run env:test`
- **开发环境**: `npm run env:dev`
- **生产环境**: `npm run env:prod`

切换环境会自动重新编译插件。

### 加载到浏览器
1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `dist` 文件夹

## 🛠 技术栈

- **React 18** - 前端框架
- **TanStack Table v8** - 数据表格组件
- **PapaParse** - CSV 数据处理
- **Webpack 5** - 构建工具
- **Chrome Extension APIs** - 浏览器扩展 API

## 📖 使用方法

1. **创建抓取器** - 访问任何网页，点击扩展图标，创建新的抓取器
2. **配置选项** - 设置抓取类型、域名、高级选项等
3. **开始抓取** - 点击"Start Scraping"开始数据抓取
4. **选择数据** - 使用表格中的复选框选择要导出的行
5. **导出数据** - 选择 CSV 或 JSON 格式导出选中的数据

## 🎯 核心改进

相比原始 Easy Scraper，我们的版本增加了：

- ✅ **行选择功能** - 使用 TanStack Table 的内置行选择
- ✅ **批量操作** - 全选/取消全选按钮
- ✅ **搜索过滤** - 实时搜索表格数据
- ✅ **排序功能** - 点击列标题进行排序
- ✅ **更好的 UI** - 现代化的界面设计
