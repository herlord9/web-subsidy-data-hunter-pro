# 🎯 Easy Scraper Clone - 使用指南

## 📋 功能说明

这个插件可以**自动识别网页上的列表数据**并抓取到表格中，比如：
- 商品列表
- 新闻列表  
- 搜索结果
- 任何重复结构的数据

## 🚀 3步开始使用

### 步骤 1️⃣: 加载插件

1. 打开 Chrome 浏览器
2. 输入地址: `chrome://extensions/`
3. 打开右上角的"开发者模式"开关
4. 点击左上角"加载已解压的扩展程序"
5. 选择文件夹: `e:\bbb-test\easy-scraper-clone\dist`

### 步骤 2️⃣: 访问你想要抓取的网页

例如打开搜索页面: `https://search.gd.gov.cn/search/all/754008`

### 步骤 3️⃣: 使用插件

1. 点击浏览器工具栏的插件图标 📍
2. 点击 **"+ 新猎手"** 创建抓取器
3. 输入名称，选择"列表抓取器"
4. 点击 **"▶ 开始抓取"**
5. 等待几秒钟，数据会显示在表格中 ✨

## 💾 导出数据

抓到数据后，你可以：
- ✅ 点击行前的复选框选择特定行
- 📊 点击"导出全部 CSV" 导出 CSV 文件
- 📄 点击"导出全部 JSON" 导出 JSON 文件
- 📋 点击"复制到剪贴板" 复制数据

## 🔍 示例场景

### 场景 1: 抓取搜索结果
1. 打开搜索结果页面
2. 创建"搜索结果"抓取器
3. 开始抓取 → 得到所有结果

### 场景 2: 抓取商品列表
1. 打开电商网站列表页
2. 创建"商品列表"抓取器  
3. 开始抓取 → 得到所有商品信息

## ⚠️ 常见问题

**Q: 插件显示空白？**  
A: 刷新页面，然后重新点击插件图标

**Q: 抓不到数据？**  
A: 确保页面有列表结构（重复的相似元素）

**Q: 想要抓特定字段？**  
A: 在创建抓取器时可以设置要提取的字段
